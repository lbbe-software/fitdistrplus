- [`danishuni`](https://lbbe-software.github.io/fitdistrplus/reference/danish.md)

  : Datasets for the FAQ
- [`endosulfan`](https://lbbe-software.github.io/fitdistrplus/reference/endosulfan.md)
  : Species Sensitivity Distribution (SSD) for endosulfan
- [`fluazinam`](https://lbbe-software.github.io/fitdistrplus/reference/fluazinam.md)
  : Species-Sensitivity Distribution (SSD) for Fluazinam
- [`fremale`](https://lbbe-software.github.io/fitdistrplus/reference/fremale.md)
  : Fictive survival dataset of a french Male population
- [`groundbeef`](https://lbbe-software.github.io/fitdistrplus/reference/groundbeef.md)
  : Ground beef serving size data set
- [`salinity`](https://lbbe-software.github.io/fitdistrplus/reference/salinity.md)
  : Species-Sensitivity Distribution (SSD) for salinity tolerance
- [`smokedfish`](https://lbbe-software.github.io/fitdistrplus/reference/smokedfish.md)
  : Contamination data of Listeria monocytogenes in smoked fish
- [`toxocara`](https://lbbe-software.github.io/fitdistrplus/reference/toxocara.md)
  : Parasite abundance in insular feral cats
